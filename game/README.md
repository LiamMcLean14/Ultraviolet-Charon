To use, run game.py. For the level select screen, use the up and down arrow keys to select a level, and enter to enter it.
In levels, hold the number keys for the lane you want to select, and press enter to 'input' that lane when the notes reach the line at the bottom of the screen
The number keys will be replaced by the camera system, and the enter key by the ultrasonic, but that is not implemented yet.
Level data is stored in the levels directory as json data. The sound will probably be changed to use wav files or something.
